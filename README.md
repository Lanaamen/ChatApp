ChitChat
----------------------------------------------------------
Chitchat är en simpel konsol-applikation där man kan chatta mellan flera terminaler eller fönster, 
alt om man använt samma eventnamn via Postman.

1. Öppna projektet med Rider eller Visual Studio.
2. Kör programmet i konsolen.
3. Öppna en separat terminal och hitta till programfilen och starta programmet även där.
4. Skriv in namn på båda fönsterna och chatta med dig själv.

----------------------------------------------------------
Kommandon i chatten

Programmet är väldigt simpelt och efter att man skrivit in sitt namn så kan man fortsätta att skriva
meddelanden som då skickas till chatten. Skriver man in 'quit' stängs istället programmet ner.
